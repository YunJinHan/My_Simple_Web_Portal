Scrapy Setting
=============
Crontab Setting
-------------
1. 터미널에서 EDITOR=vim crontab -e 실행
2. 다음 Code 작성
<pre><codei>
MAILTO = '' // 해당 예약어 수행 결과를 받아보려면 해당 계정이름을 적어줌
PATH=/usr/local/bin
LC_CTYPE="utf-8" // 한글 인코딩 문제
* * * * * * cd crawling folder 절대 경로 && scrapy crawl community
// 매 분 실행. 앞의 * 로 crontab 실행 빈도를 결정해준다.
</code></pre>
3. 완료

3. 완료
